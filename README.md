# LearnLingo

## 📌 Project Description

**LearnLingo** is a web application designed to help users find and book language teachers. The platform allows students to search for teachers based on various criteria, while teachers can create profiles to offer their services.

## 🎯 Key Features

- User authentication and database management via **Firebase**.
- Filtering and searching for teachers by language, price, rating, and experience.
- Viewing teacher profiles with detailed information.
- Booking lessons with teachers and managing reservations.
- Leaving reviews and ratings for teachers.

## 🔧 Tech Stack

### **Frontend:**

- React + Vite
- React Router
- Styled-components (or another CSS library)
- Zustand/Redux (if needed)
- Axios (for API requests)

### **Backend & Database:**

- Firebase Authentication (for user login & registration)
- Firebase Firestore (for storing user and teacher data)
- Firebase Storage (for profile images, if needed)

### **Other:**

- ESLint + Prettier (code formatting)
- Jest + React Testing Library (testing)
- **CI/CD**:
  - **Frontend deployed on Netlify**
  - **Firebase used for authentication and database**
